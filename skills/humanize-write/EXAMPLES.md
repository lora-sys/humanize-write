# 工作流示例

## 示例：比赛计划

### 用户输入（写手窗口）

```
写一份大学生创新创业比赛计划书的市场分析章节。
受众：评委（已有创业基础知识）
目的：清晰呈现市场规模、竞争格局、我们的机会

参考材料：
- 市场规模：260亿元（2024年数据）
- 主要玩家：头部3家占据60%份额
- 我们的差异化：技术门槛高、应用场景新

规则：见 RULES.md
```

### V0输出（AI草稿）

> [AI生成了包含"fascinating market opportunity"的版本]

### 裁判审查结果

```
1. 第一段用了"fascinating market opportunity" — 违反禁用词规则。建议删掉。
2. "随着市场的不断发展" — 泛泛套话，建议删除，直接说市场规模。
3. 第三段用了"这就是我们计划发挥作用的地方" — 元话语，建议删除。
4. 重复使用了"此外"作为过渡词两次 — 建议换用其他过渡。
```

### V1输出（修订版）

> [基于裁判意见修订后的版本，移除了禁用词和套话]

### V2（人类润色后）

> [用户注入了个人的分析视角、调整了段落顺序、加入了只有他知道的行业观察]

---

## 核心心法

**第一版回答只是V0，不是你要接受的草稿。**
**你才是稿件真正的主人。**
