# 工作流示例

## 完整迭代示例：比赛计划市场分析章节

### 用户输入（写手窗口）

```
场景：商业计划
主题：大学生创新创业比赛市场分析章节
受众：评委（已有创业基础知识）
目的：清晰呈现市场规模、竞争格局、我们的机会
口语温度偏好：3-4

参考材料：
- 市场规模：260亿元（2024年数据）
- 主要玩家：头部3家占据60%份额
- 我们的差异化：技术门槛高、应用场景新

规则：见 RULES.md
```

---

### V0 输出（AI草稿）

> 近年来，随着数字化转型的不断深入，市场规模达到了260亿元，展现出fascinating的增长潜力。头部三家企业占据了60%的市场份额，这不仅仅是数字上的优势，更是行业地位的体现。
>
> 我们的产品不仅仅是技术上的突破，更是解决用户痛点的方案。在当今快节奏的商业环境中，我们的产品能够seamlessly integrate到现有工作流程中，帮助企业解锁潜在价值。
>
> 综上所述，不难发现这个市场存在着巨大的机会...

---

### Moderator 审查结果

```
## 口语温度值
2/10 — 过于正式僵硬，有明显AI套话感

## 过度口语化问题
无（当前问题是AI残留，不是过度口语化）

## AI 套话残留问题
1. 「fascinating」 — 禁用词，直接移除
2. 「这不仅仅是X，更是Y」 — 禁用句式
3. 「seamlessly integrate」 — 禁用词+翻译痕迹
4. 「解锁潜在价值」 — 禁用词「unlock the potential」的中文版
5. 「综上所述，不难发现」 — 元话语+空洞概括

## 风格平衡判断
⚠️ 仍有 AI 残留，风格过于工整和空洞
```

---

### 裁判审查结果

```
1. 第一段用了「fascinating」— 违反禁用词规则，建议删除或替换为具体描述
2. 第一段「展现出XX的增长潜力」— 空洞，建议删除，直接说数据
3. 「这不仅仅是技术上的突破，更是解决用户痛点的方案」— 禁用句式，删除
4. 「在当今快节奏的商业环境中」— 泛泛套话，建议删除
5. 「能够帮助企业解锁潜在价值」— 禁用词组合，建议删除或替换为具体收益
6. 「综上所述，不难发现这个市场存在着巨大的机会」— 空洞收尾，建议改为具体结论
```

---

## 快速对照表：口语化 vs AI感

| 程度 | 感觉 | 示例 |
|---|---|---|
| 过于口语化 | 像在发微信 | 「这个方案贼棒！」 |
| 正常口语感 | 像在认真写信/说话 | 「我觉得这个方案的核心优势是...」 |
| 正常专业感 | 像专业文档 | 「该方案的核心优势在于...」 |
| 过于AI感 | 空洞工整僵硬 | 「该方案不仅仅是...更是...」 |

**V2 的目标是「正常口语感」到「正常专业感」之间，根据场景浮动。**

---

## 核心心法

1. **第一版回答只是V0，不是你要接受的草稿**
2. **Moderator 是平衡器** — 防止去AI味用力过猛
3. **迭代直到满意** — 不要接受不完整的V1，说出具体问题
4. **你才是稿件真正的主人** — AI 是工具，最终判断在你
